# A ToDo App Created in React using Redux

## Technologies Used :

1. React
2. Redux
3. React-Redux
4. Material Ui

## Features:

1. Adding Todo in real time
2. Updating a Todo List
3. Mark as Completed
4. Delete a todo list
5. Resetting the todo app
6. Filters: (Sort)
  - Show all
  - Show Active todos
  - Show only completed Todos
  - Delete Completed Todos
